BOTONES CSS BÁSICOS

Los siguientes son botones básicos con diferentes estilos CSS, y un estado hover simple, listos para usarlos de forma gratuita en cualquier sitio web. El primer ejemplo, posee solo una clase, los demás poseen subclases, Ej: class=”btn-basico.b1 ”. Con esto se puede ahorrar código en la hoja de estilo, parea no repetir los parámetros de las clases que se pueden reutilizar con las subclases.

Botón 1
HTML
<a class=”btn-basico” href=”#”>Botón 1</a>
CSS
.btn-basico {
display: inline-block;
padding: 20px 40px;
float: left;
background-color: #0094cc;
color: white;
font-size: 18px;
text-decoration: none; }

.btn-basico.b1:hover {
background-color: #007dab; }
Botón 2
HTML
<a class=”btn-basico b1” href=”#”>Botón 2</a>
CSS
.btn-basico.b1 {
padding: 10px 30px;
background-color: #616161;
font-size: 14px;
line-height: 16px; }

.btn-basico.b1:hover {
background-color: #474747; }
Botón 3
HTML
<a class=”btn-basico b2” href=”#”>Botón 3</a>
CSS
.btn-basico.b2 {
border-radius: 10px;
background-color: #786091; }

.btn-basico.b2:hover {
background-color: #60457d; }
